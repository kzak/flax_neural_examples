## Install

```sh
# Install uv
# see: https://docs.astral.sh/uv/getting-started/installation/#installation-methods
curl -LsSf https://astral.sh/uv/install.sh | sh

#
# Install and activate this project
make install
source .venv/bin/activate
```


## Execute

```sh
make jupyter
```
